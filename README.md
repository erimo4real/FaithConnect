# BETHEL CHURCH Website

A professional React-based church website featuring sermons, events, donations, gallery, blog, prayer requests, small groups, and more.

## Features

- **Home** - Hero slider, stats counter, featured ministries, testimonials, newsletter signup
- **Sermons** - Filter by speaker, year, and media type
- **Events** - Event listings with registration
- **Live Stream** - Simulated live service
- **Donations** - One-time and recurring giving options
- **Gallery** - Image gallery with lightbox
- **Blog** - Blog posts with categories
- **Prayer Request** - Submit prayer requests
- **Small Groups** - Browse and join small groups
- **Staff Directory** - Meet the team
- **Contact** - Contact form and map
- **Shop** - Merchandise store with cart

## Tech Stack

- React 18
- Vite
- React Router
- Tailwind CSS
- React Icons

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

## Project Structure

```
src/
├── components/     # Reusable components
├── pages/          # Page components
├── data/           # Static data (sermons, events, blog posts)
├── App.jsx         # Main app with routing
└── main.jsx        # Entry point
```
